## Описание проекта

Необходимо разработать решение, которое позволит персонализировать предложения постоянным клиентам, чтобы увеличить их покупательскую активность. Так же разработать модель, которая предскажет вероятность снижения покупательской активности. Выделить сегмент покупателей, проанализировать его и предложить, как увеличить его покупательскую активность.
